Rebuilding LEGO-xtal: from seed structures to 1,700 sp²-allotropes

Reproducible tutorial · datasets + configs

1) Environment

conda create -n legoxtal python=3.11
conda activate legoxtal
pip install -r requirements.txt

2) Seeds & targets

Download the 25 seed structures and local-environment JSON…

3) Generation & screening

python run_generate.py --seeds data/seeds --cfg configs/sp2.yaml

4) Relaxation pipeline (ALIGNN-FF → MACE → VASP)

python relax.py --engine alignnff --then mace --vasp
